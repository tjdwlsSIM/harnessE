# /fix

Use this command when fixing a bug.

## User Request

The text after `/fix` is the bug description.

Example:

```txt
/fix 로그인 후 메인 페이지로 이동하지 않는 문제를 해결해줘

로그인 후 메인 페이지로 이동하지 않는 문제를 해결해줘

Summary:
- What was fixed.

Cause:
- Explain the likely cause.

Changed files:
- file 1: what changed
- file 2: what changed

How to test:
- Step 1
- Step 2
- Step 3

Risks or follow-up:
- Anything that should be checked later.
