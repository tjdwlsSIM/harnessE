# /refactor

Use this command when refactoring existing code.

## User Request

The text after `/refactor` is the refactor request.

Example:

```txt
/refactor App.jsx에 몰려있는 코드를 pages와 components로 분리해줘
App.jsx에 몰려있는 코드를 pages와 components로 분리해줘

Summary:
- What was refactored.

Changed files:
- file 1: what changed
- file 2: what changed

Behavior changes:
- State whether behavior changed or not.

How to test:
- Step 1
- Step 2

Risks or follow-up:
- Anything that should be checked later.
