# Checklist

Before completing any AI coding task, check:

## Code Safety

- [ ] Did not modify unrelated files
- [ ] Did not delete existing functionality
- [ ] Did not expose secrets or API keys
- [ ] Did not hard-code environment-specific URLs

## Architecture

- [ ] Components are reusable
- [ ] API logic is separated from UI
- [ ] File names are clear
- [ ] Folder structure follows the project convention

## Quality

- [ ] No obvious syntax errors
- [ ] No unnecessary duplicate code
- [ ] Loading and error states are handled where needed
- [ ] Final summary includes changed files and test steps
